# INTRODUCCION
pacman es un juego donde controlas es un personaje que come capsulas.
El personaje principal tiene que escapar de 4 fantasmas :

- fantasma rojo
- fantasma verde
- fantasma azul
- fantasma amarillo

## controles 
- Arriba
- Abajo
- Derecha
- Izquierda

## compilar
Para ejecutar el proyecto, ejevcutar el comando
>make run
